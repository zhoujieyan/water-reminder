# 💧 水宝提醒 - 用户使用指南

欢迎使用**水宝提醒**，这是一款专为办公室打工人设计的趣味喝水助手！本应用采用渐进式Web应用（PWA）技术，让你可以像原生应用一样安装到手机主屏幕，享受离线可用、定时提醒、成就激励等丰富功能，轻松养成健康喝水习惯。

---

## ✨ 核心功能简介

### 1. 智能定时提醒
- **自定义提醒间隔**：可设置每隔多少分钟提醒一次喝水（默认60分钟）
- **多模态提醒**：支持浏览器通知 + 声音提示，确保不漏提醒
- **灵活操作**：收到提醒后可选择“喝了一杯”、“跳过”或“10分钟后提醒”

### 2. 趣味卡通界面
- **拟人化水杯角色**：可爱的“水宝”会根据你的喝水进度变换表情（😟😐😊😄🎉）
- **动态水位显示**：杯中水位随喝水进度实时上升，视觉直观
- **活泼动画**：点击记录喝水时有弹跳动画、水花溅起效果

### 3. 喝水记录与进度追踪
- **一键记录**：点击“+1杯”按钮快速记录喝水
- **实时统计**：显示今日已喝杯数、目标杯数、完成百分比
- **进度条**：直观展示当日目标完成情况

### 4. 环境智能联动
- **天气感知**：自动获取当地温度、湿度数据
- **智能推荐**：根据天气状况调整推荐喝水量（高温干燥天建议多喝）
- **实时展示**：在“今日天气”卡片中显示当前天气和推荐水量

### 5. 成就激励系统
- **多样成就**：包含连续打卡、单日达标、累计喝水等成就类型
- **徽章收集**：解锁成就可获得专属徽章奖励
- **进度追踪**：查看已解锁成就和待解锁成就的完成条件

### 6. 历史数据与健康简报
- **历史记录**：查看每日、每周、每月的喝水趋势图表
- **周报生成**：自动生成每周健康简报PDF，包含统计图表和个人建议
- **数据导出**：所有喝水记录保存在本地，保护隐私

### 7. 完整的PWA特性
- **添加到主屏幕**：像原生应用一样安装，图标出现在手机桌面
- **离线可用**：即使没有网络也能正常使用核心功能
- **自动更新**：Service Worker确保应用始终保持最新版本

## 📱 安装到主屏幕详细步骤

### iOS 设备（Safari 浏览器）
1. 打开 **Safari** 浏览器，访问水宝提醒应用网址
2. 点击底部工具栏的 **分享按钮**（方形箭头图标）
3. 在分享菜单中向下滚动，找到 **“添加到主屏幕”** 选项
4. 点击后可以修改应用名称（默认为“水宝提醒”）
5. 点击右上角 **“添加”** 按钮
6. 返回主屏幕，即可看到水宝提醒应用图标，点击即可全屏使用

### Android 设备（Chrome 浏览器）
1. 打开 **Chrome** 浏览器，访问水宝提醒应用网址
2. 点击右上角 **菜单按钮**（三个竖点）
3. 选择 **“添加到主屏幕”** 选项
4. 确认应用名称和图标，点击 **“添加”**
5. 返回主屏幕，找到水宝提醒应用图标，点击即可全屏使用

### 安装注意事项
- **网络要求**：首次访问需要网络连接以下载应用资源
- **存储空间**：应用约占用 200KB 存储空间
- **权限请求**：首次使用时会请求通知权限，请允许以接收提醒

## 🎮 基本使用方法

### 1. 首次使用设置
- **允许通知**：应用会请求通知权限，请点击“允许”以接收喝水提醒
- **天气定位**：首次使用天气功能时会请求位置权限（可选）
- **基础目标**：默认每日喝水目标为8杯（约1600ml），可根据需要调整

### 2. 记录喝水
- **快捷记录**：点击首页中间的 **“+1杯”** 按钮，立即记录一杯水
- **动画反馈**：记录成功后会看到水杯弹跳动画和水花效果
- **进度更新**：页面上的进度条、水位和统计数据会实时更新

### 3. 查看历史记录
- 点击首页 **“历史记录”** 按钮
- 查看日历视图，了解每日喝水情况
- 切换周视图、月视图，分析喝水趋势
- 图表展示：可视化呈现喝水数据变化

### 4. 成就系统
- 点击首页 **“成就”** 按钮
- 查看已解锁的成就徽章
- 了解待解锁成就的完成条件
- 成就解锁时会收到弹窗庆祝通知

### 5. 每周健康简报
- 点击首页 **“周报”** 按钮
- 系统自动生成本周喝水统计报告
- 报告包含：总杯数、达标天数、趋势图表、个性化建议
- 支持保存为PDF文件，方便分享或打印

### 6. 个性化设置
- 点击首页 **“设置”** 按钮
- **提醒间隔**：调整喝水提醒的频率（30-120分钟）
- **声音开关**：开启或关闭提醒音效
- **天气功能**：启用或禁用天气智能推荐
- **数据管理**：查看存储空间、清除缓存、重置数据

### 7. 提醒文案预览
- 点击首页 **“文案预览”** 按钮
- 模拟不同时间、天气、成就状态下的提醒文案
- 了解水宝的个性化提醒风格（幽默、科普、鼓励等）

## ❓ 常见问题解答（FAQ）

### Q1: 为什么收不到喝水提醒通知？
- **检查通知权限**：在手机设置中确保已允许浏览器发送通知
- **浏览器设置**：确保浏览器没有阻止该网站的通知
- **勿扰模式**：检查手机是否处于勿扰模式或静音状态
- **应用状态**：确保应用在前台或后台运行（PWA支持后台提醒）

### Q2: 离线时可以使用哪些功能？
- **完全可用**：所有核心功能均可离线使用，包括：
  - 记录喝水
  - 查看历史记录
  - 查看成就
  - 查看已缓存的数据
- **有限功能**：以下功能需要网络连接：
  - 天气数据更新
  - 推送通知（需网络连接发送）
  - 新版本检查

### Q3: 我的喝水数据安全吗？会泄露吗？
- **本地存储**：所有数据都保存在你设备的 IndexedDB 数据库中
- **隐私保护**：数据不会上传到任何服务器，完全本地处理
- **备份建议**：如需更换设备，建议先导出历史数据

### Q4: 天气数据不准确或无法获取怎么办？
- **手动刷新**：点击天气卡片右下角的“🔄 刷新天气”按钮
- **位置权限**：确保已授予浏览器位置权限（非必需，有默认城市）
- **备用方案**：如无法获取实时天气，应用会使用最近缓存的数据
- **关闭天气**：可在设置中关闭天气功能，使用固定推荐水量

### Q5: 如何调整每日喝水目标？
- 进入 **“设置”** 页面
- 找到 **“每日目标”** 设置项
- 输入你希望的杯数（每杯约200ml）
- 系统会根据天气情况在此基础上下浮动推荐

### Q6: 连续打卡天数是怎么计算的？
- **连续打卡**：指连续每天至少喝一杯水
- **中断规则**：如果某天没有喝水记录，连续天数会重置为0
- **查看方式**：在成就页面可以看到当前的连续打卡天数

### Q7: 如何清除所有数据重新开始？
- 进入 **“设置”** 页面
- 找到 **“数据管理”** 部分
- 点击 **“清除所有数据”** 按钮
- **注意**：此操作不可逆，所有喝水记录和成就将永久删除

### Q8: 应用图标可以更换吗？
- **图标固定**：应用图标是预设的卡通水杯图案，暂不支持自定义
- **主题颜色**：应用采用蓝绿色健康色调，与整体风格统一
- **名称修改**：在添加到主屏幕时可以自定义显示名称

### Q9: 为什么有时打开应用比较慢？
- **首次加载**：第一次访问需要下载所有资源，约需3-5秒
- **后续使用**：安装到主屏幕后，启动速度接近原生应用
- **网络状况**：检查网络连接，部分资源可能需从缓存加载
- **设备性能**：老旧设备可能运行较慢，建议关闭其他应用

### Q10: 如何反馈问题或建议？
- **应用内反馈**：暂不支持应用内直接反馈（独立PWA限制）
- **联系开发**：如有技术问题，可通过部署平台联系管理员
- **功能建议**：欢迎提出新功能想法，我们会考虑在后续版本中加入

## 🔧 技术支持与故障排除

### 浏览器兼容性
- **推荐浏览器**：Chrome 60+、Safari 12+、Edge 79+、Firefox 60+
- **移动系统**：iOS 12+、Android 8+
- **PWA支持**：确保浏览器支持 Service Worker 和 Web App Manifest

### 常见故障解决
1. **应用无法安装到主屏幕**
   - 确保使用推荐浏览器
   - 检查网址是否以 HTTPS 开头（安全连接必需）
   - 清除浏览器缓存后重试

2. **通知不工作**
   - 检查系统级通知设置
   - 在浏览器设置中重新授权通知权限
   - 确保应用没有被浏览器休眠

3. **数据丢失**
   - 检查是否清除了浏览器数据
   - 确认是否在不同设备间使用（数据不跨设备同步）
   - 定期导出重要数据备份

### 版本更新
- **自动更新**：应用安装后会自动在后台更新
- **手动更新**：如需强制更新，可卸载后重新安装
- **版本号**：在应用设置页面底部可查看当前版本
