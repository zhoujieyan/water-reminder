# 图标资源目录

此目录用于存放水宝提醒应用的图标资源文件。

## 所需图标尺寸

根据 `manifest.json` 配置，需要以下尺寸的PNG图标：

- 72x72像素
- 96x96像素  
- 128x128像素
- 144x144像素
- 152x152像素
- 192x192像素
- 384x384像素
- 512x512像素

## 设计规范

- **格式**：PNG格式，透明背景
- **风格**：趣味卡通风格，以水杯角色“水宝”为主体
- **色彩**：蓝色绿色主色调，与整体设计风格一致
- **圆角**：建议使用适当的圆角，增强亲和力

## 占位说明

当前为开发阶段，使用占位图标。正式版本需要设计并替换为实际图标。

## 生成建议

可以使用在线图标生成工具或设计软件（如Figma、Adobe Illustrator）创建适配多尺寸的图标集。
